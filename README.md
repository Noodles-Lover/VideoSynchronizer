# VideoSynchronizer

这是一个让你可以在一个网页上同步观看两个视频（例如一边看动漫，一边看反应视频）的小工具。你可以自由移动视频的位置，或是调整它们的大小。支持移动端。
（主要是我自己想用来看那些同步视听的视频，才开发的这个小工具）

访问 -> https://noodles-lover.github.io/VideoSynchronizer/

## 🎮 操作

- **移动视频**：按住视频**顶部边缘**，就可以像拖动窗口一样把它移到任何地方。
- **调整大小**：点击并拖动视频**右下角**的位置，就可以放大或缩小视频。
- **快捷工具栏**：点击视频窗口或移动视频时，右侧会出现一个**自动隐藏的工具栏**：
  - **📌 置顶**：将该视频置于最前方。
  - **🔲 全屏**：将该视频切换为全屏显示。
  - **🔆 透明度**：垂直滑动条可以独立调整该窗口的不透明度（最低 20%）。
  - **⚙️ 快速设置**：点击齿轮可以直接为该窗口更换视频或调整开始时间。

## ⚙️ 设置说明

除了视频窗上的快捷工具栏，右上角的**主设置面板**提供更全面的控制：

1. **放上视频**：
   - **YouTube**：在输入栏里贴上 YouTube 网址。支持带有时间戳的网址（如 `?t=120`）。
   - **本地文件**：点击输入框旁的「上传」按钮，选择本地视频。
   - **直链**：贴入视频文件直链（如 `.mp4`）。
   - **设置时间**：在秒数输入框填入数字，指定视频开始播放的时间点。
2. **同时播放**：
   - 对好时间后，点击底部的「▶ 同时播放」按钮，两个视频会同步跳转并开始。
3. **全局调整**：
   - **按钮位置**：调整主设置按钮在左上角或右上角。
   - **强制置顶**：设置某个视频始终保持在最前。
   - **全屏显示**：切换全屏状态。
4. **面板透明度**：
   - 调整主设置面板的不透明度，避免遮挡视线。

## 📝 备注

- **支持格式**：
  - **YouTube** 视频。
  - **本地视频**（.mp4, .webm, .ogg 等浏览器支持的格式）。
  - **视频直链**（部分网站的直链可能由于服务器端的防盗链策略导致无法加载播放）。
- **触屏友好**：在手机或平板上也可以直接用手指拖拽操作喔！
- **播放问题**：如果视频播放有问题可以尝试多种浏览器（如 Chrome, Firefox, Safari 等）。
